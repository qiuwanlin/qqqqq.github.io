this & arguments

  • this 就是 call 第第一个参数。call 的其他参数统称为 arguments。

  • this 是隐藏的第一个参数，且必须是对象。

  • this 为什么必须是对象,因为 this 就是函数与对象之间的羁绊。

bind

~~~
var view = {
element:$(‘#div’)
bindEvents:function(){
this.element.onclick = function(){
this.onClick() //浏览器调这个函数,this是触发事件的元素(div)
}
},
onClick:function(){}
}
那怎么办，只能把this记下来，让this是view
bindEvents:function(){
var _this = this
this.element.onclick = function(){
_this.onClick()
}
},
之后发明了bind:
bindEvents:function(){
this.element.onclick = this.onClick.bind(this)
},
