Posted on 2018-08-02 | In es6

字面量加强

  • 字面量就是很直观的一种量，直接写出来的量就是字面量，比如字符串和{}、[]等。非字面量就是构造出来的量，比如new Object()。
  1. 更安全的进制字面量：
    二进制是以0b或0B开头。八进制是以0o开头。
  2. 字符串支持Unicode：
    es6新增了几个专门处理4字节码点的函数：
  • String.fromCodePoint(): 从Unicode码点返回对应字符
  • String.prototype.codePointAt():从字符返回对应的码点
    “”.codePointAt(0).toString(16)
  1. 正则表达式字面量添加Unicode支持，添加y标记，支持粘滞匹配。

Symbol

  • ES5有六种数据类型：
    string number bool undefined null object(array function data regex不算，属于对象)
  • ES6增加了 symbol
  • symbol很好理解但似乎并不常用。

迭代

迭代器

  • 迭代会返回一个对象，其中提供了一个next()方法，用来返回序列中的下一项，方法中包含两个属性，done value。
  • 迭代器对象一旦被创建，就可以反复调用next()。

生成器
