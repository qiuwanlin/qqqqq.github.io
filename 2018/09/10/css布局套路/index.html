css浮动布局套路

Posted on 2018-09-10 | In css

适用面广，支持IE5。
使用套路：
给子元素加上 样式：float:left
给父元素加上 类：clearfix

用float做平均布局：
可以用.xxx:nth-child(4n+1){}这样的语法，但是ie不支持，
所以要擅用负margin噢~

  • 用来布局的标签就不要有内容。
  • 高度最好给里面，让它自己撑外面。

用flex做平均布局：

  1. 不要用 space-between
  2. 用负margin，或者用nth-child(),也可用margin配合calc。
  3. 记得在 父元素 上加 overflow: hidden;
