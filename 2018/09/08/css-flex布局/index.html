css flex布局

Posted on 2018-09-08 | In css

flex container的属性

flex-direction（方向）

默认 row：横向
row-reverse ：反向
columnn ：竖向
column-reverse :竖反向

flex-wrap（换行）

默认 nowrap :不换行
wrap :自动换行

  • flex-flow是上面那俩的简写

justify-content（主轴方向对齐方式）

默认 flex-start: 元素和容器的左端对齐。
flex-end: 元素和容器的右端对齐。
center: 元素在容器里居中。
space-between:元素之间保持相等的距离。
space-around:元素周围保持相等的距离。

align-items（侧轴对齐方式）

默认 stretch: 元素被拉伸以填满整个容器。
flex-start: 元素与容器的顶部对齐。
flex-end: 元素与容器的底部对齐。
center: 元素纵向居中。

baseline: 元素在容器的基线位置显示。（用得少）

align-content （多行/列内容对齐方式）（用得少）

flex item的属性

flex-grow（增长比例（空间过多时））

假设有三个子元素：

  • flex-shrink （收缩比例（空间不够时））
    这个属性一般不用，用法和上一个类似。

flex-basis（默认大小）

这个属性一般不用。
不写就是原始大小，写了就按你写的来显示。

  • flex属性是以上三个属性的简写。
    写法如 flex:1 2 100px

order（顺序）

改变元素展示顺序。
代替双飞翼布局的原始套路。

align-self（自身对齐方式）

对抗父元素的align-items属性。
属性值与其相同。
