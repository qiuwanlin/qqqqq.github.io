Array 的用法

Array 对象是用于构造数组的全局对象。

  1. var a = Array(3)
    生成一个长度为3的数组。但是数组里没有下标。
  2. var a = Array(3,3)
    生成一个长度为2的数组:[‘0’:3,’1’:3]
  • 但对于number string boolean来说，不加new返回基本类型，加了new返回对象。
    对object(object array function)来说加不加new都一样。

常用数组API

forEach

a.forEach() //这是个需要接受函数的API
a.forEach(function(){} ) //这个函数必须接受两个参数 value key

map

map和forEach几乎一样，只是map有返回值，不仅是遍历数组，还会收集对数组操作的结果。

filter

用法和map很像，过滤作用，返回符合条件的数组。

sort

a.sort()排序自身 a会变，这些里只有只有只有它会改自身。
a.sort(function(x,y){return x-y})
x-y 从小到大排列 y-x 从大到小排列

join

a = [‘1’,’2’,’3’]
能把数组变字符串。就像 1 + ‘’//“1” a + ‘’ //“1,2,3”
a.join(‘q’) //“1q2q3”
a.join() //“1,2,3” 不传参就默认用逗号相隔，和a.toString()结果一样。

concat

a = [1,2,3] b = [4,5,6]
a+b //“1,2,34,5,6” 相当于a.toString()+b.toString()
a.concat(b) //[1,2,3,4,5,6] 不改变ab
常用来复制数组：
var c = a.concat([]) //c:[1,2,3] c===a false

reduce

a.reduce(function(sum,n){
return sum + n
},0)
给个初始值作为最后一个参数，第一个参数是一个函数，函数接受两个参数，第一个参数是之前的结果，第二个参数是下一次要加的数字，函数返回一个东西作为下次遍历用到的结果。
reduce可以表示map filter。
